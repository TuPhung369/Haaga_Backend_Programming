Project Overview

This documentation provides a detailed overview of the TuPhung Project, a comprehensive full-stack application built with React and TypeScript for the frontend and Spring Boot for the backend. The project includes various features such as authentication, user management, chat functionality, kanban boards, calendars, and AI assistants.

Key Features

  • Multi-factor Authentication - Secure login with TOTP and email OTP options
  • OAuth2 Integration - Login with Google, Facebook, and GitHub
  • User Management - Role-based access control and permissions
  • Real-time Chat - WebSocket-based messaging system
  • Kanban Board - Task management with drag-and-drop functionality
  • Calendar - Event scheduling and management
  • AI Assistants - Intelligent virtual assistants for various tasks
  • Language AI - Speech recognition and processing capabilities
